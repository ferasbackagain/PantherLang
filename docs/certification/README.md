# PantherLang Certification

## Certification Tracks

### 1. PantherLang Foundations
- **Prerequisites**: None
- **Coverage**: Academy Lessons 01-05, Book Chapters 01-05
- **Skills**: Variables, types, expressions, control flow, functions, conversions
- **Assessment**: 
  - 40-question multiple choice exam
  - Practical: CLI calculator project
- **Estimated Study Time**: 40 hours
- **Capstone**: Personal Diary CLI

### 2. PantherLang Developer
- **Prerequisites**: Foundations or equivalent
- **Coverage**: Academy Lessons 01-12, Book Chapters 01-12
- **Skills**: Data structures, stdlib, security, web platform, database, AI basics
- **Assessment**:
  - 60-question exam (mixed format)
  - Practical: Web + SQLite application
- **Estimated Study Time**: 80 hours
- **Capstone**: Library Management API or URL Shortener

### 3. PantherLang Professional
- **Prerequisites**: Developer or equivalent
- **Coverage**: Academy Lessons 01-18, Book Chapters 01-18
- **Skills**: Full-stack development, AI integration, cross-platform, security
- **Assessment**:
  - 100-question comprehensive exam
  - Capstone project (choose any advanced capstone)
- **Estimated Study Time**: 160 hours
- **Capstone**: AI Chat with Security or Secure File Vault

### 4. PantherLang Security Specialist
- **Prerequisites**: Foundations
- **Coverage**: Academy Lessons 01-08, 14-15; Book Chapters 01-08, 14-15
- **Skills**: Security diagnostics, crypto functions, secure coding patterns
- **Assessment**: Security audit of provided code
- **Capstone**: Secure File Vault

### 5. PantherLang Web Developer
- **Prerequisites**: Developer
- **Coverage**: Academy Lessons 01-10, 12; Book Chapters 01-10, 12
- **Skills**: Web routing, API design, SQLite, JSON, security middleware
- **Assessment**: Build a REST API with documentation
- **Capstone**: Library Management API or URL Shortener

### 6. PantherLang AI Engineer
- **Prerequisites**: Developer
- **Coverage**: Academy Lessons 01-11; Book Chapters 01-11
- **Skills**: AI providers, Agent/SecureAgent, RAG, prompt injection defense
- **Assessment**: Deploy a secure AI agent
- **Capstone**: AI Chat with Security

### 7. PantherLang Educator
- **Prerequisites**: Professional
- **Coverage**: All materials
- **Skills**: Teaching, curriculum design, assessment creation
- **Assessment**: Create an original lesson with verification
- **Capstone**: New Academy lesson contribution

All certifications are valid for 2 years. Recertification requires:

- Submitting a new capstone project
