# PantherLang AST Definitions

R3 Batch 2 Part 3.1 defines the official PantherLang AST model.

## Core Nodes

- ProgramNode
- MainBlockNode
- WebBlockNode
- ApiBlockNode
- AiBlockNode
- TestBlockNode
- BlockNode

## Statements

- PrintStatement
- ReturnStatement
- ExpressionStatement
- VariableDeclaration
- AssignmentStatement
- IfStatement
- WhileStatement
- RouteStatement

## Expressions

- IdentifierExpression
- StringLiteral
- NumberLiteral
- BooleanLiteral
- NullLiteral
- UnaryExpression
- BinaryExpression
- CallExpression
- MemberExpression
- ObjectLiteral
- ArrayLiteral

Parser implementation begins in R3 Batch 2 Part 3.2.
