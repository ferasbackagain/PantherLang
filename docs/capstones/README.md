7 capstone projects across CLI, Web, Data, Security, and AI tracks.

## Projects

| # | Project | Level | Track | Skills |
|---|---------|-------|-------|--------|
| 1 | Personal Diary CLI | Beginner | CLI | Filesystem, strings, arrays |

## Getting Started

Each capstone includes:
- **Specification** (.md) — requirements, rubric, verification steps
- **Solution** (.pan) — reference implementation

```bash
# Run a solution
python -m cli.panther_cli run docs/capstones/solutions/url-shortener.pan
```
