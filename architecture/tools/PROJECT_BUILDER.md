# Panther Project Builder — Phase 1.15

## Purpose
The Project Builder creates standard Panther project structures.

## Phase 1.15 Scope
- Create a basic Panther application folder.
- Generate README.md.
- Generate app.panther.
- Generate panther.project.json.
- Prepare for future `panther new` command.
