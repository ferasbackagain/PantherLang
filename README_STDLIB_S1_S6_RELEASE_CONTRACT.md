# PantherLang Stdlib S1-S6 Release Contract

Run from the PantherLang repository root:

```bash
unzip -o pantherlang_stdlib_s1_s6_release_contract.zip
chmod +x bootstrap_stdlib_s1_s6_release_contract.sh
./bootstrap_stdlib_s1_s6_release_contract.sh
```

This package adds stable API documentation, a runnable PantherLang example, Linux/Windows runners, tests, engineering report, manifest, and then runs full regression.
