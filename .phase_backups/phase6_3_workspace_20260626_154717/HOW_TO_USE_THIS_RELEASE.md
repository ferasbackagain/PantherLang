# How to Use This Release

This release is a full clean edition. Do not unzip it inside `language/`.

Recommended:

```bash
cd ~/pantherlang
unzip PantherLang_Developer_Edition_v0_5.zip
cd PantherLang_Developer_Edition_v0_5
```

Test:

```bash
bash scripts/test.sh
```

Run:

```bash
bash scripts/run_store.sh
```

Push to GitHub:

```bash
git add .
git commit -m "PantherLang v0.5 - Developer Edition"
git push
```

If you want to replace the old repository content, copy the contents of this folder into your Git repo root, or use this folder as the new official root.
