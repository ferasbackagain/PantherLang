# PantherLang JSON Parser Example

Demonstrates PantherLang JSON operations:
- `json_encode()` — convert object to JSON string
- `json_decode()` — parse JSON string to object/array

## Run

```bash
panther run examples/json_parser/main.pan
```

## Expected Output

Encodes a Panther object to JSON, decodes it back, works with arrays
and nested objects, accesses elements by index and key.
