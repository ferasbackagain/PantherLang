# Calculator Example

Demonstrates PantherLang arithmetic and recursion:
- Integer arithmetic (`+`, `-`, `*`, `/`, `%`, `**`)
- Comparison operators (`>`, `<`, `==`)
- Recursive function (`factorial`)
- String conversion (`string()`)
- `if` / `else` control flow

## Run

```bash
panther run examples/calculator/calc.pan
```

## Expected Output

```
PantherLang Calculator
a = 42, b = 7
a + b = 49
a - b = 35
a * b = 294
a / b = 6
a % b = 0
a ** 2 = 1764
x = 10, y = 3
x > y: true
x == y: false
x < y: false
factorial(5) = 120
factorial(7) = 5040
```
