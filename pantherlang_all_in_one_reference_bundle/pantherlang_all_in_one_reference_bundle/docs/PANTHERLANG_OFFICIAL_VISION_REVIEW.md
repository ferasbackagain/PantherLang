# PantherLang Official Vision Review

PantherLang is the programming language foundation of the Panther ecosystem. The calculator is not the business product; it is a reference application that forces the language to prove the primitives required by real software.

## Why the reference app matters

A reference app prevents fake progress. Expressions, variables, input, output, conditions, functions, runtime execution, CLI integration, diagnostics, tests, reports, manifests, and backups must all work together.

## Rule

No Feature Without Proof: implementation, Kali execution, regression tests, fixes, manifest, report, backup, then completion.
