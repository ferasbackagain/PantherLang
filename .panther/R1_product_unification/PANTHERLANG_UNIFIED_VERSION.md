# PantherLang Unified Version Contract

## Target Version

`1.0.0`

## Release Name

`PantherLang Developer Edition v1.0.0`

## Policy

All public PantherLang product components should align to the same user-facing version:

- CLI
- Compiler
- Runtime
- Debug Adapter
- Language Server
- Toolchain
- Package Manager
- Standard Library
- VS Code Extension
- Documentation
- Release Packages

Historical backups and archived release packages must not be rewritten.
