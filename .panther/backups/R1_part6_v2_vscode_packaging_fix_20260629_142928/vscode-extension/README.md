# PantherLang

Official PantherLang language support, developer tooling, and debug adapter integration for Visual Studio Code.

## Features

- PantherLang syntax support for `.pan` and `.panther`
- PantherLang commands
- Project creation workflow
- Run current PantherLang file
- PantherLang Debug Adapter integration
- Dark-blue PantherLang product identity

## Requirements

Install PantherLang CLI and ensure it is available on your PATH.

## Getting Started

1. Install the extension.
2. Open a `.pan` file.
3. Run `PantherLang: Doctor`.
4. Create a new project with `PantherLang: New Project`.
5. Run or debug your PantherLang program.
