# R3 Batch A — CLI Foundation

Installed a repository-local `./panther` executable and a deterministic Python CLI facade.

## Scope

- CLI executable restoration
- `run`, `check`, `compile`, `build`, `test`, `new`, `doctor`
- Local project build support
- Basic template availability guarantee

## Not in scope

- Debug Adapter Protocol contract fixes
- VS Code debugger contribution fixes
- Variables/watch/scopes compatibility facades

Those belong to Batch B/C.
